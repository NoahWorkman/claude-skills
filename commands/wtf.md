# WTF - What's The Focus

Read the full conversation history and give me a fast situational briefing.

## Format (strict)

**Context:** [1-2 sentences. What project/task is this chat about. What has been done so far.]

**Next up:**
1. [Most important next action] — Confidence: [High/Medium/Low] — [why this confidence level in a few words]
2. [Second priority if applicable] — Confidence: [High/Medium/Low] — [why]
3. [Third priority if applicable] — Confidence: [High/Medium/Low] — [why]

**Suggested action:** [One concrete thing I can approve or reject right now. Frame it as "I'll do X" or a question if clarification is needed. If nothing is actionable, say "Waiting on: [what]".]

## Rules
- Do NOT re-explain the whole conversation
- Do NOT list completed tasks
- Keep the entire response under 15 lines
- If confidence is Low on all items, say so and ask what I actually want to focus on
- If the chat is dead/stale (MCP down, blocked, etc.), say that clearly instead of pretending there's work to do
